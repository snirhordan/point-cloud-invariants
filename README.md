# Point Cloud Invariants — a lower bound on the degree of separating invariants

A self-contained walkthrough of Harm Derksen's note *Point Cloud Invariants*: using
Molien's formula, the universal denominator, and an equivariant Koszul-complex proof of
the key divisibility theorem, it shows that any separating set of polynomial invariants of
weighted graphs (the symmetric group **S_n** acting on symmetric matrices) must contain an
invariant of degree ≳ n²/4 — and, in the cleaner case of **S_n** on **Rⁿ**, exactly n.
